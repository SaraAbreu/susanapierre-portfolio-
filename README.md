# SUSANA_PIERRE // AI & AUTOMATION PORTFOLIO

## Stack de Arquitectura
- **Core:** Next.js / React
- **Estética:** Tailwind CSS (Industrial Design System)
- **Despliegue sugerido:** Vercel o Netlify

## Concepto de Diseño
Interfaz de telemetría de alta gama basada en el rigor técnico. 
Uso de paleta de metales preciosos oxidados (Old Gold / Antique Silver) 
sobre fondo Obsidian con textura de ruido analógico.

## Scripts de Ejecución
1. Instalar dependencias: `npm install`
2. Modo desarrollo: `npm run dev`
3. Construcción producción: `npm run build`
